# Денис & Лидия — свадебное приглашение

Файлы для GitHub Pages лежат в корне архива: `index.html`, `style.css`, `script.js`, папка `assets` и файл `.nojekyll`.

Что важно при загрузке на GitHub:
1. Загружайте содержимое архива в корень репозитория, а не папку целиком.
2. Не переименовывайте папки `assets/images` и `assets/music`.
3. В репозитории должны быть видны файлы `index.html`, `style.css`, `script.js` на одном уровне.

Музыка запускается автоматически при загрузке сайта, а если браузер блокирует звук без действия пользователя, скрипт повторит запуск после первого касания, скролла или нажатия клавиши.
