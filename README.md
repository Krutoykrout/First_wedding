# Свадебный сайт Дениса и Лидии

Готовый статический сайт для GitHub Pages.

## Что внутри

- `index.html` — разметка сайта.
- `style.css` — весь дизайн: чёрно-белый минимализм, карточки, жемчужные детали.
- `script.js` — таймер до свадьбы, кнопка музыки, отправка анкеты гостя.
- `assets/images/dress-men.jpg` — фото дресс-кода для мужчин.
- `assets/images/dress-women.jpg` — фото дресс-кода для девушек.
- `assets/music/song.mp3` — песня на сайте.
- `.nojekyll` — файл для корректной публикации на GitHub Pages.

## Куда приходят ответы гостей

Форма отправляет ответы только на почту:

`ekenotova@yandex.ru`

На сайте нет Telegram, SMS, MAX или других вариантов ответа.

Важно: отправка сделана через бесплатный сервис FormSubmit, потому что GitHub Pages — это статический хостинг без своего сервера. При первой тестовой отправке на почту `ekenotova@yandex.ru` может прийти письмо от FormSubmit для подтверждения адреса. Нужно открыть письмо и подтвердить адрес, после этого ответы гостей будут приходить на почту.

## Как загрузить на GitHub и получить ссылку

1. Распакуйте ZIP-архив.
2. Зайдите на GitHub и создайте новый репозиторий, например `wedding-denis-lidia`.
3. Загрузите в репозиторий все файлы и папки из распакованного архива.
4. Важно: `index.html`, `style.css` и `script.js` должны лежать в корне репозитория, не внутри лишней папки.
5. Откройте репозиторий на GitHub.
6. Перейдите в `Settings` → `Pages`.
7. В блоке `Build and deployment` выберите:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - папка: `/root`
8. Нажмите `Save`.
9. Через 1–3 минуты на этой же странице появится ссылка вида:

`https://ваш-логин.github.io/wedding-denis-lidia/`

Это и есть ссылка на сайт. Её можно отправлять гостям.

## Что можно менять

### Музыка

Чтобы заменить песню, положите новый MP3-файл в папку:

`assets/music/`

и назовите его:

`song.mp3`

### Фото дресс-кода

Чтобы заменить фото, положите новые изображения в папку:

`assets/images/`

и назовите их:

- `dress-men.jpg`
- `dress-women.jpg`

### Тексты, время, адреса

Тексты, время и адреса меняются в файле `index.html`.
