# Свадебный сайт Дениса и Лидии

Готовый статический лендинг-приглашение для GitHub Pages.

## Что внутри

- `index.html` — структура сайта.
- `style.css` — дизайн в стиле референса: молочный шёлк, тёмные волны, жемчужные фоновые детали, мобильная премиум-композиция.
- `script.js` — открывающий экран, музыка, анимированный таймер, плавные появления блоков, лёгкий параллакс и отправка анкеты.
- `assets/images/couple-sunset.jpg` — фото пары на закате.
- `assets/images/dress-men.jpg` и `assets/images/dress-women.jpg` — изображения для дресс-кода.
- `assets/music/song.mp3` — музыка для сайта.
- `.nojekyll` — файл для корректной публикации на GitHub Pages.

## Важно по форме

Точный адрес получателя не выводится на странице. Он подставляется скриптом при отправке формы.

## Как загрузить на GitHub Pages

1. Распакуйте ZIP-архив.
2. Создайте новый репозиторий на GitHub.
3. Загрузите все файлы и папки из архива в корень репозитория.
4. Откройте `Settings` → `Pages`.
5. Выберите `Deploy from a branch`, ветку `main`, папку `/root`.
6. Сохраните настройки и дождитесь появления ссылки.

## Что можно менять

Музыка меняется заменой файла `assets/music/song.mp3`.

Фото меняются заменой файлов в папке `assets/images/` с сохранением имён.

Тексты, время и адреса меняются в `index.html`.


## Обновление responsive

Эта версия адаптирована не только под телефон, но и под планшеты/ПК: на широком экране лендинг раскрывается в полноценный desktop-layout с двухколоночными секциями, широкими блоками, сохранёнными анимациями, таймлайном и параллакс-фото.
